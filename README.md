## Batakpedia - The Interactive Web Encyclopedia of the Batak People
Batakpedia is an interactive website that showcases everything about the Batak people of Sumatra, Indonesia, starting from the sub-group identities, language, history and culture. Each element in this website has their own uniqueness.
